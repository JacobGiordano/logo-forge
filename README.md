# Logo Forge

**[→ Open Logo Forge](https://jacobgiordano.github.io/logo-forge/)**

A browser-based raster-to-SVG converter built for logo cleanup. Upload a PNG or JPG, tune the trace settings, and download production-ready SVGs and PNGs in a full set of sizes — favicons, light/dark variants, OG images, and more.

## Features

- Auto or manual thresholding with bias control
- Despeckling, hole preservation, and corner cleanup before tracing
- Curve fitting and path simplification controls for smoother, lighter SVGs
- Auto-detects whether to invert (dark logo on light bg vs. light on dark)
- Live update mode — re-traces as you adjust sliders
- Undo/redo trace history (Cmd+Z / Cmd+Shift+Z, up to 20 steps)
- Processed-mask preview plus richer SVG diagnostics
- SVG preview on white, black, and transparency-check backgrounds
- One-click download: SVG variants, individual PNGs, or a full ZIP

## Usage

1. Drop or upload an image
2. Adjust trace settings
3. Click **Trace to SVG**
4. Download from the SVG or Exports tab

No install, no server, no dependencies — runs entirely in the browser.
